# Specification

## Summary
**Goal:** Build a responsive marketing/landing website for “Uttah Electronics” with key brand sections and an electronics-themed 3D animated experience.

**Planned changes:**
- Create a responsive landing-page layout with top navigation and section anchor links.
- Add sections: Hero, Featured Categories, Featured Products (static sample catalog with at least 6 items), About, and Contact.
- Implement a 3D hero scene using Three.js via React Three Fiber with continuous animation.
- Add at least one additional animated 3D element or section transition effect beyond the hero scene.
- Apply a consistent, modern visual theme across the site that avoids a blue/purple-dominant palette.
- Build a Contact form UI (Name, Email, Message) with a non-error submit confirmation interaction.

**User-visible outcome:** Visitors can browse a polished Uttah Electronics landing page, view featured categories and sample products, experience animated 3D visuals, and submit a message via the contact form.
